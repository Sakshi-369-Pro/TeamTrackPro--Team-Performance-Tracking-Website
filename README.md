# 🚀 TeamTrack Pro  
### AI-Powered Team Performance & Project Intelligence Platform

An advanced full-stack platform designed to manage teams, track projects, analyze performance, and optimize resumes using AI-driven insights.

---

## 🔥 Key Highlights

- 👥 Role-based dashboards (Manager & Team Member)
- 📊 Real-time project & task tracking
- 🤖 AI-powered smart candidate selection
- 💬 Team & project chat system
- 📈 Performance analytics & leaderboards
- 🧠 Skill tracking & auto-detection UI
- 📄 ATS-friendly resume builder
- 🌙 Dark/Light mode with persistent themes

---

## 🧩 Problem Solved

- Lack of visibility in team performance  
- Poor task tracking and accountability  
- No structured skill intelligence system  
- Difficulty in resume optimization for ATS  
- Weak collaboration tools in teams
